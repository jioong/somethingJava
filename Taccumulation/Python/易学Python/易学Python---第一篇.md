# 易学 Python #

Python 非常适合用于编写下列程序：

	- Web 开发。
	- 网络通信。
	- 图形用户界面。
	- 执行操作系统任务的脚本。
	- 游戏。
	- 数据处理。
	- 商业用于程序。

## Python 标准库 ##

标准库随着 Python 一起安装，，是一个庞大的程序代码集，**可用于执行常见任务。**如：查找和迭代文件、处理用户输入、下载和分析网页以及访问数据库。  
编写程序时能充分利用标准库，通常可节省时间、减少输入量和 bug 。

### 其他库 ###

并非只能使用随 Python 安装的库，还可以根据需要轻松的下载并安装其他库。大多数库都有**安装程序或安装脚本，通常不能直接复制到 Python 安装目录下的库文件夹中。**

### 使用库 ###

1. 包含一切  
可将整个库都包含在脚本中  

    from os import *

    access("myfile.txt")    # 使用时

将读取整个 os 模块，并将其放到脚本中。  
**缺点：**

	- 将包含很多怪异的函数。
	- 后导入的模块中的函数覆盖先导入的模块中的函数。
	- 很难记住特定函数来自哪个模块。导致难以维护。

2. 包含模块  

	import os  

使用时需要使用：

	os.access("myfile.txt")

3. 只包含所需的部分  

    from os import access

### 库到底是什么 ###

库可以包含标准 Python 支持的任何变量：变量、函数、类以及加载库时应运行的 Python 代码。库不受任何限制，可包含在 Python 中任何合法的东西。
